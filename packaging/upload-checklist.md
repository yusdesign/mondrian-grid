# Release Checklist

## Before Release
- [♥] Test on Windows with Inkscape 1.4.3
- [ ] Test on Linux (if possible)
- [ ] Test on macOS (if possible)
- [♥] Update version number in code
- [ ] Update CHANGELOG.md
- [ ] Take fresh screenshots
- [♥] Create example SVG files

## Package Contents
- [♥] mondriangrid_v3.py (main extension)
- [♥] mondriangrid_v3.inx (GUI definition)
- [♥] README.md (documentation)
- [♥] LICENSE (GPL v2+)
- [♥] INSTALL.md (installation guide)
- [♥] CHANGELOG.md (version history)
- [♥] Optional: examples/ directory
- [♥] Optional: screenshots/
