# Changelog

## [1.0.0] - 2024-01-15
### Added
- Initial release of Mondrian Grid Generator
- Complete parameter set: canvas, grid, colors, composition
- 5 color palettes: Classic, Modern, Grayscale, Primary, Pastel
- Composition controls: Distribution, Balance, Randomness
- Advanced options: random seed, grouping, background
- Full documentation and examples

### Technical
- Compatible with Inkscape 1.0+
- Uses standard inkex API
- No external dependencies
- Clean SVG output
